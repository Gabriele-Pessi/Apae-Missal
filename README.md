## Sobre o projeto
Este projeto tem como objetivo divulgar informações sobre a APAE de Missal, facilitar a comunicação com a comunidade, captar voluntários e receber doações de forma prática e segura. O site possui páginas para:

- Sobre a APAE
- Como ajudar
- Contato
- Visualização de doações e voluntariado

---

## Tecnologias utilizadas
- **HTML5** para a estrutura do site  
- **CSS3** para o design e layout  
- **Git & GitHub** para versionamento e hospedagem

## Como contribuir
Se você deseja contribuir com o projeto:

1. Faça um fork deste repositório.  
2. Crie uma branch para sua modificação (`git checkout -b feature/nova-funcionalidade`).  
3. Faça commit das suas alterações (`git commit -m "Descrição da modificação"`).  
4. Faça push para a sua branch (`git push origin feature/nova-funcionalidade`).  
5. Abra um Pull Request no repositório original.
